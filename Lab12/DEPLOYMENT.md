# Deployment Information

**Platform:** Render (Blueprint)

## Public URL

```
https://day12-agent-arzd.onrender.com
```

**Demo UI:** https://day12-agent-arzd.onrender.com/ (paste `AGENT_API_KEY` in settings)

## Setup summary

1. Push repo to GitHub
2. Render → **New +** → **Blueprint** → connect repo
3. Set `OPENAI_API_KEY` when prompted (optional)
4. Deploy → copy URL and `AGENT_API_KEY` from Environment tab

Full steps: [06-lab-complete/RENDER.md](06-lab-complete/RENDER.md)

## Environment Variables

| Variable | Required | Notes |
|----------|----------|--------|
| `AGENT_API_KEY` | Yes | Auto-generated by Render Blueprint |
| `REDIS_URL` | Yes | Linked from `day12-redis` service |
| `ENVIRONMENT` | Yes | `production` |
| `OPENAI_API_KEY` | Optional | Paste your OpenAI key for real LLM |
| `RATE_LIMIT_PER_MINUTE` | No | Default `10` |
| `MONTHLY_BUDGET_USD` | No | Default `10.0` |

## Test Commands

### Health Check

```bash
curl https://YOUR-SERVICE.onrender.com/health
```

### API Test

```bash
curl -X POST https://YOUR-SERVICE.onrender.com/ask \
  -H "X-API-Key: YOUR_AGENT_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"user_id": "test", "question": "Hello"}'
```

### Conversation

```bash
curl -X POST https://YOUR-SERVICE.onrender.com/ask \
  -H "X-API-Key: YOUR_AGENT_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"user_id": "test", "question": "What did I just say?"}'
```
